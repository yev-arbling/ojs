# Shopify Integration Guide

> **Audience:** Retailers and Shopify partners adding OJS structured data to product pages. If you build an enrichment pipeline or run validation in CI, see [docs/domains/](../domains/README.md) and the Python transformer docs instead.

This guide shows how to add Open Jewelry Schema v1.0 data to a Shopify store using metafields and JSON-LD. OJS data on product pages is readable by AI shopping agents (ChatGPT, Perplexity, Google AI Mode, Copilot), Google Rich Results, and any tool that understands JSON-LD.

## Overview

OJS v1.0 uses a **modular structure**: the root record has four required modules (`identity`, `commerce`, `media`, `audit`) plus optional domain modules (`metals`, `stones`, `setting`, `sizing`, `style`, etc.). On Shopify, this maps to:

- **Metafields** under namespace `ojs` — one metafield per OJS field/module you want to expose
- **JSON-LD `<script>` tag** — generated by a Liquid snippet that reads your metafields and outputs a v1.0-compliant JSON-LD block
- **Python transformer** — for pipeline-generated records, use `to_shopify_metafields()` from `src/python/ojs/transformers/shopify.py` to go directly from a `JewelryProduct` object to a list of metafields

## Step 1: Create metafield definitions

In Shopify Admin → Settings → Custom data → Products, create definitions under namespace `ojs`.

Essential fields to start:

| Metafield key | Type | v1.0 path | Example value |
|---|---|---|---|
| `ojs.product_type` | Single line text | `product_type` | `ring` |
| `ojs.metal_type` | Single line text | `metals.compositions[0].type` | `gold` |
| `ojs.metal_purity_karat` | Decimal number | `metals.compositions[0].purity_karat` | `14` |
| `ojs.metal_color` | Single line text | `metals.compositions[0].color` | `yellow` |
| `ojs.stone_species` | Single line text | `stones.stones[0].species` | `diamond` |
| `ojs.stone_origin` | Single line text | `stones.stones[0].origin_type` | `natural` |
| `ojs.stone_cut` | Single line text | `stones.stones[0].cut` | `round_brilliant` |
| `ojs.stone_carat` | Decimal number | `stones.stones[0].carat` | `1.02` |
| `ojs.stone_clarity` | Single line text | `stones.stones[0].clarity_grade` | `VS1` |
| `ojs.stone_color_grade` | Single line text | `stones.stones[0].color_grade` | `F` |
| `ojs.setting_type` | Single line text | `setting.styles[0].setting_type` | `prong` |
| `ojs.style_era` | Single line text | `style.era` | `contemporary` |
| `ojs.ring_size_us` | Decimal number | `sizing.ring_size.us_ca` | `7.0` |
| `ojs.ring_resizable` | Boolean | `sizing.ring_resizable` | `true` |

> **v1.0 field path changes from v0.1:**
> - `metal.type` → `metals.compositions[0].type`
> - `metal.purity` → `metals.compositions[0].purity_karat` (numeric) or `purity_fineness` (integer)
> - `stones[0].cut` → `stones.stones[0].cut`
> - `setting.type` → `setting.styles[0].setting_type`
> - `sizing.ring_size_us` → `sizing.ring_size.us_ca`
> - `offers[].price` → `commerce.offers[0].price.amount` (string)

## Step 2: Add the JSON-LD Liquid snippet

Create `snippets/ojs-jsonld.liquid` in your theme:

```liquid
{% assign ojs_metal_type = product.metafields.ojs.metal_type.value %}
{% assign ojs_stone_species = product.metafields.ojs.stone_species.value %}
{% assign ojs_product_type = product.metafields.ojs.product_type.value | default: 'other' %}

<script type="application/ld+json">
{
  "@context": "https://openjewelryschema.org/v1/context.jsonld",
  "@type": "Product",
  "product_type": {{ ojs_product_type | json }},
  "identity": {
    "sku": {{ product.selected_or_first_available_variant.sku | json }},
    "title": {{ product.title | json }},
    "description": {{ product.description | strip_html | truncate: 5000 | json }},
    "brand": { "name": {{ shop.name | json }} }
  },
  "commerce": {
    "offers": [{
      "price": {
        "amount": {{ product.price | money_without_currency | json }},
        "currency": {{ cart.currency.iso_code | json }}
      },
      "availability": {% if product.available %}"in_stock"{% else %}"out_of_stock"{% endif %},
      "url": {{ shop.url | append: product.url | json }},
      "condition": "new",
      "target_countries": ["US"],
      "seller_name": {{ shop.name | json }},
      "seller_url": {{ shop.url | json }}
    }]
  },
  "media": {
    "images": [
      { "url": {{ product.featured_image | img_url: 'master' | prepend: 'https:' | json }}, "role": "primary", "is_primary": true }
      {% for image in product.images offset: 1 limit: 9 %}
      ,{ "url": {{ image | img_url: 'master' | prepend: 'https:' | json }}, "role": "additional", "is_primary": false }
      {% endfor %}
    ]
  },
  "audit": {
    "ojs_version": "1.0.1",
    "created_at": {{ 'now' | date: '%Y-%m-%dT%H:%M:%SZ' | json }},
    "source_system": "shopify"
  }
  {% if ojs_metal_type != blank %}
  ,"metals": {
    "compositions": [{
      "type": {{ ojs_metal_type | json }},
      "primary": true
      {% assign purity = product.metafields.ojs.metal_purity_karat.value %}
      {% if purity != blank %}
      ,"purity_karat": {{ purity }}
      {% endif %}
      {% assign mcolor = product.metafields.ojs.metal_color.value %}
      {% if mcolor != blank %}
      ,"color": {{ mcolor | json }}
      {% endif %}
    }]
  }
  {% endif %}
  {% if ojs_stone_species != blank %}
  ,"stones": {
    "stones": [{
      "species": {{ ojs_stone_species | json }},
      "origin_type": {{ product.metafields.ojs.stone_origin.value | default: "natural" | json }}
      {% assign scut = product.metafields.ojs.stone_cut.value %}
      {% if scut != blank %}
      ,"cut": {{ scut | json }}
      {% endif %}
      {% assign scarat = product.metafields.ojs.stone_carat.value %}
      {% if scarat != blank %}
      ,"carat": {{ scarat }}
      {% endif %}
    }]
  }
  {% endif %}
  {% assign setting_type = product.metafields.ojs.setting_type.value %}
  {% if setting_type != blank %}
  ,"setting": {
    "styles": [{ "style_id": "main", "setting_type": {{ setting_type | json }} }]
  }
  {% endif %}
  {% assign ring_size = product.metafields.ojs.ring_size_us.value %}
  {% if ring_size != blank %}
  ,"sizing": {
    "ring_size": { "us_ca": {{ ring_size }} }
    {% assign resizable = product.metafields.ojs.ring_resizable.value %}
    {% if resizable != blank %}
    ,"ring_resizable": {{ resizable }}
    {% endif %}
  }
  {% endif %}
  {% assign style_era = product.metafields.ojs.style_era.value %}
  {% if style_era != blank %}
  ,"style": {
    "era": {{ style_era | json }},
    "design_styles": [],
    "motifs": [],
    "aesthetic_tags": []
  }
  {% endif %}
}
</script>
```

Include the snippet in `sections/main-product.liquid`:

```liquid
{% render 'ojs-jsonld' %}
```

### Programmatic via Pydantic (pipeline approach)

If you generate OJS records in Python, use the Shopify transformer directly:

```python
from ojs.models import JewelryProduct
from ojs.transformers.shopify import to_shopify_metafields

product = JewelryProduct.model_validate(data)
metafields = to_shopify_metafields(product)
# metafields is a list of { namespace, key, value, type } dicts
# Push via Admin API or Shopify Python client
```

The `to_shopify_metafields()` function in `src/python/ojs/transformers/shopify.py` does a full round-trip-safe mapping from the v1.0 model to Shopify metafields under the `ojs` namespace.

## Step 3: Batch-update metafields via Admin API

To populate OJS metafields for existing products programmatically:

**Set a single metafield:**
```bash
curl -X POST \
  "https://YOUR-STORE.myshopify.com/admin/api/2024-10/products/PRODUCT_ID/metafields.json" \
  -H "X-Shopify-Access-Token: YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "metafield": {
      "namespace": "ojs",
      "key": "product_type",
      "value": "ring",
      "type": "single_line_text_field"
    }
  }'
```

**Bulk update via GraphQL (recommended for large catalogs):**
```graphql
mutation productUpdate($input: ProductInput!) {
  productUpdate(input: $input) {
    product {
      id
      metafields(first: 10) {
        edges { node { namespace key value } }
      }
    }
  }
}
```

Variables:
```json
{
  "input": {
    "id": "gid://shopify/Product/PRODUCT_ID",
    "metafields": [
      { "namespace": "ojs", "key": "product_type", "value": "ring", "type": "single_line_text_field" },
      { "namespace": "ojs", "key": "metal_type", "value": "gold", "type": "single_line_text_field" },
      { "namespace": "ojs", "key": "stone_species", "value": "diamond", "type": "single_line_text_field" }
    ]
  }
}
```

## Step 4: Validate

After adding the snippet, view page source and copy the JSON-LD object. Validate with the OJS CLI validator:

```bash
# Install globally
npm install -g @openjewelryschema/validate

# Save JSON-LD content to a file, then:
ojs-validate product.json
```

Or use Google's Rich Results Test on your product URL.
